# Roadmap

## v0.2 Native Agent Harness

Terminal cwd awareness, structured ripgrep workspace search, a bounded
read-only tool loop, command palette, workspace trust, and activity/prompt
polish.

## v0.3 Codex CLI Adapter

Explicit configuration and approval, visible execution, temporary worktree
isolation where appropriate, and diff import through existing patch safety.

## v0.4 MCP / ChatGPT Connector Experiment

Opt-in connector boundaries without browser session hijacking, cookie access,
hidden tokens, or implicit execution.

Native Browser, LSP integration, SSH workspaces, and background jobs remain
future work.
